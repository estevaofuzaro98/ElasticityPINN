### Usage
To get started with **ElasticityPINN**, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/estevaofuzaro98/ElasticityPINN.git
   ```
2. Navigate to the package directory:
   ```bash
   cd ElasticityPINN
   ```
